================================================================================
FORM FIELD TYPES SUPPORTED
================================================================================

In submit_report.html and template rendering:
  - text: standard text input
  - number: number input with step 0.01
  - checkbox: toggle with label
  - select: dropdown from field.options array
  - textarea: multiline text input

All fields are rendered dynamically from template.fields JSON array.
Field structure: { key, label_ar, type, options (for select), ... }

JAVASCRIPT FEATURES

   - Real-time grade calculation (A-E)
